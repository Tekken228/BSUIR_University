void * memcpy( void * destptr, const void * srcptr, size_t num );
Функция memcpy копирует num байтов первого блока памяти, на который ссылается указатель srcptr, во второй блок памяти, 
на который ссылается указатель destptr.

void * memmove( void * destptr, const void * srcptr, size_t num );
Функции memcpy() и memmove() в языке программирования C используются для копирования блоков памяти из одного места в другое.
memcpy() не проверяет на перекрывание исходной и целевой областей памяти. Если эти области перекрываются, результат работы функции будет 
неопределенным.

memmove() предусматривает возможность перекрывающихся областей памяти и обрабатывает их безопасно. Она копирует блок памяти во 
временный буфер, а затем вставляет его в целевую область памяти. Это делает memmove() более медленной, но более надежной функцией для 
копирования блоков памяти, особенно в случае, когда перекрывающиеся области памяти необходимы.

Функция копирует srcptr в destptr.

Копирует количество байт

char * strcat( char * destptr, const char * srcptr );
Объединение строк. Функция добавляет копию строки srcptr в конец строки destptr.

char* strncat(char* s1, const char* s2, int n)
Объединение строк. Функция добавляет n-ое количество байт строки srcptr в конец строки destptr.

int memcmp( const void * memptr1, const void * memptr2, size_t num );
Функция сравнивает первые num байтов блока памяти указателя memptr1 с первыми num байтами блока памяти memptr2. 
Возвращаемое значение 0 если блоки равны, и значение отличное от 0, если  блоки не равны.

int strcmp( const char * string1, const char * string2 );
Эта функция сравнивает символы двух строк, string1 и string2. Начиная с первых символов функция strcmp сравнивает поочередно каждую 
пару символов, и продолжается это до тех пор, пока не будут найдены различные символы или не будет достигнут конец строки.

int strnccmp(const char* s1, const char* s2, int n)
Сравнивает первые n символов строк

char* mystrtok(char* str, const char* delim)
Функция strtok выполняет поиск лексем в строке string. Последовательность вызовов этой функции разбивают строку string на лексемы, 
которые представляют собой последовательности символов, разделенных символами разделителями.

void * memset( void * memptr, int val, size_t num );
Функция memset заполняет num байтов блока памяти, через указатель memptr. Код заполняемого символа передаётся в функцию через параметр val.

const char* streerror(int errnum)
Функция strerror() возвращает сообщение об ошибке, связанное с номером ошибки.
